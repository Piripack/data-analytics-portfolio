# MKD Pharmacies Portfolio Page

This folder contains a portfolio-safe project page built from the `Contemporary Web Applications` module submission.

Files:

- `index.html` - main project page
- `styles.css` - local styling
- `lightbox.js` - click-to-expand image viewer
- `images/` - only the verified diagrams and screenshots used on the page

Source evidence used:

- `AE1_REPORT_FINAL_10300471.docx`
- `AE2_Pharmacy_Implementation_Report.docx`
- `AE2_MKD_Pharmacies_Submission.zip`

Why this is merged:

- `AE1` contains the system design and architecture work
- `AE2` contains the implemented Next.js application and verification screenshots
- together they form one stronger portfolio project than two weaker separate entries
