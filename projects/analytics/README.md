# Portfolio Site

This folder is ready to add to a GitHub repository as a static portfolio section.

## Files

- `index.html` - landing page
- `excel-superstore.html` - Excel dashboard project
- `python-superstore.html` - Python EDA and cleaning project
- `tableau-superstore.html` - Tableau dashboard project
- `student-ml.html` - student-performance clustering and KNN project
- `styles.css` - shared styling
- `lightbox.js` - click-to-expand image viewer
- `images/` - only the images used by the pages

## Notes

- All image references are local to this folder.
- Open `index.html` to browse the portfolio.
- The content is based on extracted material from the coursework report, rewritten into portfolio format.
