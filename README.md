# Data Analytics Portfolio

This repository contains a small static portfolio site built from selected analytics and BI coursework projects.

## Projects

- [Portfolio Home](./index.html)
- [Excel Superstore Dashboard](./excel-superstore.html)
- [Python EDA and Data Cleaning](./python-superstore.html)
- [Tableau Sales and Profit Dashboard](./tableau-superstore.html)
- [Student Performance Clustering and KNN Classification](./student-ml.html)

## Stack

- HTML
- CSS
- JavaScript
- Excel
- Python
- Tableau

## Site Structure

- `index.html` - portfolio landing page
- `excel-superstore.html` - Excel dashboard project page
- `python-superstore.html` - Python EDA and cleaning project page
- `tableau-superstore.html` - Tableau dashboard project page
- `student-ml.html` - machine learning project page
- `styles.css` - shared styling
- `lightbox.js` - image expand viewer
- `images/` - locally bundled project screenshots

## GitHub Pages

If GitHub Pages is enabled for this repository, `index.html` should be used as the site homepage.

The `README.md` appears on the repository page on GitHub, but it does not replace `index.html` as the GitHub Pages landing page.

Expected Pages URL after enabling GitHub Pages:

- `https://piripack.github.io/data-analytics-portfolio/`

## Notes

- Only the images used by the portfolio pages are included.
- Image expansion is handled client-side with `lightbox.js`.
- The project wording is portfolio-focused rather than coursework-formatted.
